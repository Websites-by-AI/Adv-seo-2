مدل استقرار

Vercel و Docker ورودی‌های متفاوت دارند

Vercel Serverless

Vercel فایل Frontend و Flask API را از ساختار استاندارد تشخیص می‌دهد.

public/index.html api/index.py requirements.txt vercel.json PUBLIC_BASE_URL=https://PROJECT.vercel.app PDF_LINK_SECRET=LONG_RANDOM_SECRET
در Vercel فایل server.py به‌صورت سرور دائمی اجرا نمی‌شود؛ api/index.py مسئول API است.

Docker / Hugging Face

Docker یک سرور Python دائمی روی پورت 7860 اجرا می‌کند.

Dockerfile server.py app.html HOST=0.0.0.0 PORT=7860 DRY_RUN=true
برای Hugging Face نوع Space را روی Docker قرار دهید. صفحه اصلی سرور به‌صورت خودکار app.html است.
